# MySQL

在本系列文章中我们使用官方的 [test_db](https://github.com/datacharmer/test_db) 以及标准的电商库 [mysql-mall-matrix https://url.wx-coder.cn/Lmzp3](https://url.wx-coder.cn/Lmzp3)
作为我们的测试数据库。
